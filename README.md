## Deskripsi

Quick Access by Aplikasiku adalah aplikasi mobile yang dirancang untuk memudahkan pengguna dengan berbagai fitur praktis. Aplikasi ini memiliki tampilan yang bersih dan intuitif, cocok untuk kebutuhan sehari-hari. Ini merupakan tugas akhir semester saya dalam pengembangan aplikasi mobile.

## Fitur Utama

- **QR Scan**: Pindai kode QR dengan cepat dan mudah.
- **My Location**: Temukan lokasi Anda saat ini dengan akurat.
- **Web Viewer**: Buka dan jelajahi situs web langsung dari aplikasi.
- **Navigation**: Navigasi yang mudah antar fitur aplikasi.
- **About**: Informasi tentang aplikasi dan pengembang.
- **Log Out**: Keluar dari aplikasi dengan aman.

## Download dan Install

1. Klik tombol "Download APK" di atas atau unduh file `Quick Access.apk` dari bagian Releases.
2. Pastikan perangkat Android Anda mengizinkan instalasi dari sumber tidak dikenal (Pengaturan > Keamanan > Sumber Tidak Dikenal).
3. Buka file APK yang diunduh dan ikuti instruksi instalasi.
4. Setelah instalasi selesai, buka aplikasi dan nikmati fitur-fiturnya!

## Persyaratan Sistem

- Android 5.0 (Lollipop) atau lebih tinggi
- Izin akses kamera (untuk QR Scan)
- Izin akses lokasi (untuk My Location)
